# 배드엔딩 상세 진입 라우트 (메인 경로형)

아래는 **현재 story_manifest(확장 분기형)** 기준의 실제 진입 흐름입니다.

## BE_01 — 성급한 언론 대응 붕괴
**진입 조건 노드:** `C04_N2`에서 `B 성급한 언론 대응`

### 대표 라우트
`C01_N1 -> C02_N2 -> C03_N2 -> C04_N2(B) -> BE_01`

### 이야기 흐름
- 보호를 우선하던 라인이 공론장 대응을 서두르면서, 증거·보호 체계가 동시에 무너짐.
- 사건 프레임을 선점당해 이후 재판 루트가 붕괴.

---

## BE_02 — 침투 실패 / 로그 유실
**진입 조건 노드:** `C06_N2`에서 `B 추적 실패`

### 대표 라우트
`C01_N1 -> C02_N1 -> C03_N1 -> C04_N1 -> C05_N1(B 야간 침투) -> C06_N2(B) -> BE_02`

### 이야기 흐름
- 우회 침투는 성공하면 큰 이득이지만, 실패 시 로그 회수선 전체가 끊김.
- 주인공 측의 조작 입증 능력이 급감해 조기 파국.

---

## BE_03 — 과잉 노출 역풍
**진입 조건 노드:** `C08_N2`에서 `B 과잉 노출`

### 대표 라우트
`C01_N1 -> C02_N1 -> C03_N1 -> C04_N1 -> C05_N1(B) -> C06_N2(A) -> C07_N2 -> C08_N2(B) -> BE_03`

### 이야기 흐름
- 정보전 라인에서 노출 강도를 조절하지 못해, 반격 명분을 상대에게 제공.
- 여론과 법정 모두 역전당함.

---

## BE_04 — 중간심리 성급 강행
**진입 조건 노드:** `C10_N1`에서 `B 성급 강행`

### 대표 라우트
`C01_N1 -> C02_N1 -> C03_N1 -> C04_N1 -> C05_N1(A) -> C06_N1 -> C07_N1 -> C08_N1(A) -> C09_N1 -> C10_N1(B) -> BE_04`

### 이야기 흐름
- 기소선에서 속도를 무리하게 올리며 심리 단계 검증이 붕괴.
- 불완전 제출이 치명타가 되어 사건 전체가 조기 종료.

---

## BE_05 — 실명 전면공개 강행 파탄
**진입 조건 노드:** 각 루트 `C14_R_*`에서 `C 실명 전면 공개 강행`

### 공통 진입 골격
`(공통 1~12) -> C13_R_[루트] -> C14_R_[루트](C) -> BE_05`

### 루트별 표기
- `... -> C14_R_NAYEON(C) -> BE_05`
- `... -> C14_R_SIYU(C) -> BE_05`
- `... -> C14_R_HARIN(C) -> BE_05`
- `... -> C14_R_IDEN(C) -> BE_05`
- `... -> C14_R_YURA(C) -> BE_05`

### 이야기 흐름
- 신원 공개 강도를 제어하지 못해 보호선 붕괴 + 정치 역풍 동시 발생.

## BE_06 — 양립 강행 붕괴

`(공통 1~12) -> C13_R_[루트] -> ... -> C17_R_[루트](C) -> BE_06`
