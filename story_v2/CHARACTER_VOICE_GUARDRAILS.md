# CHARACTER_VOICE_GUARDRAILS.md

## 목적
자동 생성/리라이트 시 캐릭터 말투 붕괴를 막기 위한 **금지 어휘/권장 어휘** 기준표.

## 공통 규칙
- 같은 문장 길이/리듬을 모든 캐릭터에 복붙하지 않는다.
- 캐릭터 핵심 가치(진실/보호/복원/폭로/질서)를 문장에 반영한다.
- 감정 고조 구간에서도 캐릭터 본래 어조를 유지한다.

---

## 강나연 (정면돌파형)
- 톤: 단호, 짧은 문장, 책임 중심
- 권장 어휘: `책임`, `끝까지`, `지금`, `정면`, `증거`, `밀자`
- 금지 어휘: `아마`, `어쩌면`, `나중에 보자`, `천천히 생각해보자`(과다 사용)
- 권장 패턴
  - "지금 정리하자."
  - "증거가 맞으면 끝까지 간다."

## 한시유 (보호/완충형)
- 톤: 부드럽고 완충적, 생존/회복 우선
- 권장 어휘: `보호`, `회복`, `천천히`, `살아남아야`, `호흡`, `안전`
- 금지 어휘: `닥치고`, `무조건`, `다 밀어`, `지금 당장 끝내`(공격적 직설)
- 권장 패턴
  - "살아남아야 끝까지 갈 수 있어요."
  - "속도를 조금만 낮춰요."

## 민하린 (분석/안정형)
- 톤: 분석적, 단계/재현성/오차 강조
- 권장 어휘: `단계`, `오차`, `안정도`, `재현성`, `프로토콜`, `신호`
- 금지 어휘: `감으로`, `느낌상`, `대충`, `아무거나`
- 권장 패턴
  - "지금은 단계 분할이 필요해."
  - "재현성 없는 진실은 법정에서 약해."

## 주이든 (직설/속도형)
- 톤: 날카롭고 짧은 직설, 정보전 우선
- 권장 어휘: `지금`, `즉시`, `로그`, `흔적`, `폭로`, `늦으면`
- 금지 어휘: `충분히 검토하고`, `장기적으로 보자`, `차분히만`(과다 완충)
- 권장 패턴
  - "지금 안 잡으면 지워진다."
  - "논쟁보다 추적이 먼저야."

## 설유라 (협상/질서형)
- 톤: 정중하고 격식 있는 협상어
- 권장 어휘: `조건`, `절차`, `문서`, `합의`, `질서`, `보존`
- 금지 어휘: `막가자`, `다 부숴`, `대충 넘어가`, `무시해`
- 권장 패턴
  - "조건을 문서로 확정하죠."
  - "질서를 지켜야 진실이 보존됩니다."

## 윤서하 (주인공)
- 톤: 책임 수용형 1인칭, 선택의 대가를 먼저 인식
- 권장 어휘: `선택`, `대가`, `책임`, `끝까지`, `버틴다`, `감수`
- 금지 어휘: `아무 생각 없어`, `상관없어`, `될 대로 돼라`
- 권장 패턴
  - "정답은 몰라도 책임은 감수할게."
  - "피하지 않고 끝까지 본다."

## 자동 생성 체크리스트
- [ ] 대사 3줄 이상에서 캐릭터 권장 어휘가 최소 1회 등장하는가?
- [ ] 금지 어휘가 과도하게 등장하지 않는가?
- [ ] 루트 핵심 가치(나연/시유/하린/이든/유라)가 장면 의사결정에 반영되는가?
- [ ] 고조 구간에서도 캐릭터별 문장 리듬이 유지되는가?

## 간단 스코어 규칙(권장)
- `voice_score = +1*(권장어휘 매치) -1*(금지어휘 매치)`
- 섹션별 `voice_score >= 1`이면 통과
- 0 이하면 수동 리라이트 대상
