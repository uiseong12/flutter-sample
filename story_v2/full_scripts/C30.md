# C30 작가모드 통합 원고

이 장은 분기별 결과가 다음 선택의 비용으로 이어지도록 재작성되었다. 템플릿 반복 문장을 제거하고, 노드별 장면 목적을 분리했다.

## 노드 C30_R_HARIN — C30 · 루트 최종 판결 (하린 루트)
- route: HARIN (민하린)

**장면 요약**  
기자회견 백스테이지에서 서하는 직전 분기의 후속 비용을 확인하고, 이번 장의 핵심 갈등을 검증 프레임으로 정리한다.

**윤서하**  
"이번 선택도 결국 누군가의 오늘을 바꿔. 그래서 끝까지 책임지고 고를게."

**민하린**  
"신호와 잡음을 먼저 나눠."

**선택지 유지**
- (A) [진행] 루트 해피엔딩 확정 -> `HE_HARIN`
- (B) [고위험] 질서 과잉 선택 -> `BE_09`
- (C) [고위험] 진실 과잉 선택 -> `BE_10`
