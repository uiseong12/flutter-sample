# C08 확장판 풀 원고 (C08_N1)


## 톤 일관성 기준 (노드/챕터 동기화)
- 강나연: 짧고 단호한 문장 / 책임·정면돌파
- 한시유: 완충적 어조 / 보호·회복 우선
- 민하린: 분석형 어조 / 안정성·재현성
- 주이든: 직설적 어조 / 속도·폭로
- 설유라: 정중한 협상 어조 / 질서·제도 보존
- 윤서하: 선택의 대가를 먼저 인식하고 책임을 감수하는 1인칭 감정선


## 노드: C08_N1 〈대역 가설〉

### Scene 1
나연은 공격 문구를, 시유는 완충 문구를 제안했다. 둘 다 서하를 지키려 했지만 방식이 달랐다.

### Scene 2
서하는 진술서 첫 줄을 열 번 고쳤다. 단어 하나가 재판의 성격을 바꿨다.

### Scene 3
"연루 가능성"을 쓰면 전쟁이 시작되고, "합성 정황"을 쓰면 시간을 번다.

## 선택지
1. **[강한 문구]** -> `C09_N1`
2. **[완충 문구]** -> `C09_N1`


## 연출 강화 패치 (5분 체감 보강)

### Scene 4 — 감정 잔향 4
- 연출 큐: [CAM: medium -> close], [BGM: pulse_soft], [SFX: fabric_rustle]

선택 직후의 공기는 생각보다 오래 남는다. 말은 끝났는데, 표정은 아직 대답 중이다.  서하는 방금 내린 결정을 머릿속에서 한 번 더 뒤집어 본다. 돌아갈 수 없다는 사실이 오히려 발걸음을 또렷하게 만든다.
