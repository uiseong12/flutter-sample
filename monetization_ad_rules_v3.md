# Monetization & Ad Rules V3

## IAP (가챠 없음)
1. 시즌 패스(권 단위)
2. 프리미엄 선택지 토큰
3. 코스튬/CG 팩
4. 캐릭터 보이스 팩

## 무과금 완주 원칙
- 최소 1개 메인 엔딩 + 캐릭터 1루트 보장
- 과금은 속도/연출/후일담 확장에 집중

## 광고 정책 (보상형 중심)
### 허용 위치
- 노드 결과 화면
- 아르바이트 종료 화면
- 데이트 종료 화면(추가 보이스/후일담)
- 위기 구제 선택(재판 리트라이)

### 금지 위치
- 고백/이별/화해/서약 감정씬
- 엔딩 컷신 중간

## 인터스티셜 최소화
- 일일 진입 시 낮은 확률
- 설정에서 광고 제거 패스 제공

## 프리미엄 선택지 설계 원칙
- 정답 판매 금지
- 기본 선택지로도 스토리 진행 가능
- 프리미엄은 감정씬 깊이/연출 확장 제공
