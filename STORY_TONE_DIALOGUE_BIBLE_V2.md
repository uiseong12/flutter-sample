# STORY TONE & DIALOGUE BIBLE V2

## MCP 참고 요약 (novel-material)
- 공식석상/연회/궁정 장면: **격식 있는 표면 대화 + 내부 계산 독백**이 몰입에 강함.
- 긴장 장면: 짧은 단문/호흡 끊기 + 상대 반응 묘사(시선, 침묵, 손동작)로 압력 생성.
- 감정 고조: 노골적 고백보다 **간접 문장(책임/균형/판/대가)**이 캐릭터 일관성을 살림.
- 체크포인트 직전: “지금 결정하면 누군가 다친다/이득 본다” 류의 비용 암시가 효과적.

## 캐릭터 말투 기준선

### 엘리안
- 키워드: 질서, 명예, 약속, 책임
- 문장: 짧고 직설적. 감정을 숨기되 행동으로 드러냄.
- 금지: 장황한 은유, 감정 과다 표출.

### 루시안
- 키워드: 변수, 대가, 확률, 균형, 계산
- 문장: 분석형 + 은유 1개 정도. 단정 대신 조건문 활용.
- 금지: 지나친 감정 직설.

### 세레나
- 키워드: 판, 문장, 프레임, 선점, 여론
- 문장: 부드럽지만 날카롭게. 선택지를 제시하며 압박.
- 금지: 노골적 적대/감정적 폭발.

### 주인공(나)
- 초반: 방어적 관찰
- 중반: 책임 수용 + 전략 판단
- 후반: 선언형 문장 증가

## 체크포인트 전/후 감정 연출 규칙

### 체크포인트 전
- 감정 온도 상승: 침묵, 시선, 호흡 묘사
- 정보 온도 상승: “누가 이득/손해를 보는가”를 대사로 노출

### 체크포인트 선택 순간
- 선택지는 철학/전략이 충돌하도록 구성
- 각 선택지 결과는 즉시 체감: 호감/정치/로그 문장

### 체크포인트 후
- 같은 사건이라도 캐릭터 반응 톤이 달라져야 함
- 다음 대사 2~3줄 내에 “선택의 여파”를 반드시 언급

## 노드 1~5 샘플 (실사용형)

### Node 1 계약약혼
- 나: "이 약혼이 나를 지키는 울타리일지, 족쇄일지는 아직 모른다."
- 엘리안: "이건 감정이 아니다. 약속이다. 흔들리지 마라."
- 루시안: "정치적 안전장치지. 다만 안전장치도, 압력이 쌓이면 터진다."
- 세레나: "사람들은 네가 누구를 택했는지보다, 누가 널 택했는지 기억해."

체크포인트 A 전 긴장:
- 서명 직전 펜촉이 멈춘다. 복도 끝에서 군화 소리가 끊긴다.

체크포인트 A 후 반응 예:
- 질서 선택: 엘리안이 반걸음 다가와 "좋다. 이제 내 책임이다." 
- 민심 선택: 세레나가 미소로 "좋아. 프레임은 내가 잡지."

### Node 2 복도의 거래
- 세레나: "우린 적이 아니야. 다만 같은 편도 아니지."
- 루시안: "이건 협상이라기보다, 손실 최소화 계산에 가까워."
- 나: "그럼 묻지. 누구의 손실을 기준으로 계산할 건데?"

체크포인트 전 긴장:
- 촛불이 꺼지고, 대화는 속삭임으로 내려감.

### Node 3 길드 장부
- 루시안: "숫자는 거짓말하지 않아. 다만 사람은 숫자를 거짓말하게 만들지."
- 엘리안: "장부는 증거다. 책임자는 별개다. 둘 다 잡아야 끝난다."
- 세레나: "장부를 공개하는 순간, 판도 공개돼. 준비됐어?"

### Node 4 첫 의전
- 나: "오늘은 옷이 아니라, 나 자신이 시험대에 오른다."
- 세레나: "보랏빛은 색이 아니야. 선언이야."
- 엘리안: "실수는 한 번이면 충분하다. 오늘은 없다."

### Node 5 기사단장의 선택
- 엘리안: "명예를 걸겠다. 네가 뒤돌아서지 않는다면."
- 나: "돌아서지 않을게. 대신, 네가 먼저 손을 놓지 마."
- (감정 선택 시)
  - 엘리안: "…나는 네 편이다. 질서보다 먼저."
