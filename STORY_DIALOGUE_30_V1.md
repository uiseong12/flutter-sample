# 《보랏빛 규율: 은실로 짠 왕관》
## 노드 1~30 상황/대화 설계안 (V1)

> 목표: 각 노드 약 5분 플레이 체감(상황묘사 + 대화 + 선택 2회 이상)
> 형식: 노드별 상황 / 등장인물 / 진행(씬) / 선택지 체크포인트 A,B

---

## ACT 1 — 궁정 입성 (노드 1~6)

### 노드 1. 위반된 입궁
- 상황: 주인공이 ‘보랏빛 규율’을 어기고 궁정에 들어왔다는 고발이 올라온다.
- 등장인물: 주인공, 엘리안, 루시안, 세레나
- 진행
  1) 새벽 안개, 대리석 복도. “입궁 기록이 비어 있다.”
  2) 엘리안: “규율은 칼이다. 예외를 허락하면 피를 본다.”
  3) 루시안: “기록은 조작될 수 있어. 결론을 서두르지 마.”
  4) 세레나: “진실보다 먼저, 누가 이득을 보는지 보자.”
- 선택 A
  - A1 정면 해명: “내가 직접 말하겠다.”
  - A2 침묵 관찰: “누가 먼저 흔들리는지 보자.”
  - A3 엘리안에게 보호 요청
- 선택 B
  - B1 기록실 즉시 조사
  - B2 공판까지 증거 은닉 탐색

### 노드 2. 복도 끝의 거래
- 상황: 공개 충돌 직전, 비공식 협상 기회가 열린다.
- 등장인물: 주인공, 세레나(라이벌 포지션 강화)
- 진행
  1) 비밀 복도, 촛불 반사. 발소리만 남음.
  2) 세레나: “오늘 널 살리면, 내일 넌 누구 편이지?”
  3) 주인공: “편은 결과로 정해진다.”
  4) 긴장된 시선전, 체면과 이해관계 충돌.
- 선택 A
  - A1 공개적으로 맞선다
  - A2 비공개로 체면을 살려준다
- 선택 B
  - B1 동맹 가능성만 열어둔다
  - B2 단호히 선 긋기

### 노드 3. 길드의 장부
- 상황: 장부 조작 정황 발견. 경제 축이 흔들린다.
- 등장인물: 주인공, 세레나(길드 라인), 루시안
- 진행
  1) 장부실, 잉크 냄새. 찢긴 페이지 흔적.
  2) 루시안: “숫자는 거짓말하지 않아. 다만 사람이 거짓말할 뿐.”
  3) 세레나: “숫자는 칼이야. 누가 쥐느냐가 중요하지.”
  4) ‘가짜 보랏빛’ 유통 단서 발견.
- 선택 A
  - A1 장부를 압수해 공개
  - A2 일부만 숨기고 추적
  - A3 세레나에게 공동 분석 제안
- 선택 B
  - B1 왕실 회계에 즉시 보고
  - B2 길드 내부 반응 먼저 확인

### 노드 4. 첫 의전
- 상황: 정식 의전에서 주인공의 정통성을 시험.
- 등장인물: 주인공, 엘리안, 세레나, 궁정 귀족들
- 진행
  1) 황금 홀. 시선이 칼날처럼 쏟아짐.
  2) 엘리안: “오늘 한 번 비틀리면, 모두가 물어뜯는다.”
  3) 세레나: “미소를 잃지 마. 승패는 표정에서 난다.”
  4) 의전 순서 중 함정 질문 등장.
- 선택 A
  - A1 규율 원문 인용으로 돌파
  - A2 감정 호소로 민심 확보
- 선택 B
  - B1 엘리안 라인(질서 우선)
  - B2 세레나 라인(외교 우선)

### 노드 5. 기사단장의 선택
- 상황: 야간 순찰 중 습격. 보호와 진심이 교차.
- 등장인물: 주인공, 엘리안
- 진행
  1) 빗속 훈련장. 금속성 타격음.
  2) 엘리안: “다치면 안 된다. 적은 네 피를 원해.”
  3) 짧은 근접전 후 숨 고르기.
  4) 감정선이 처음 흔들림.
- 선택 A
  - A1 감사 인사
  - A2 냉정 유지
  - A3 (프리미엄) 갑옷 끈을 직접 묶어준다
- 선택 B
  - B1 기사단 보호를 수용
  - B2 단독 행동 고수

### 노드 6. 은등회의 밤
- 상황: 은등회 치료소에서 민심의 실체를 목격.
- 등장인물: 주인공, 루시안(은등회 원장 포지션), 시민들
- 진행
  1) 허브 향과 신음. 전쟁 아닌데도 상처가 많다.
  2) 루시안: “권력은 멀리 있지만 상처는 가까이 있어.”
  3) 주인공이 직접 붕대 감기.
  4) 민심과 정치의 접점 체감.
- 선택 A
  - A1 공개 지원 선언
  - A2 비밀 지원으로 안전 확보
- 선택 B
  - B1 치료소 확대 예산 약속
  - B2 가짜 약초 유통 추적 우선

## ACT 2 — 공판과 균열 (노드 7~12)

### 노드 7. 가짜 보랏빛
- 상황: 가짜 규율 문서 유통망 확인.
- 등장인물: 주인공, 세레나, 루시안
- 핵심: 증거 확보 vs 잠복 추적
- 선택 A: 즉시 보고 / 은닉 후 추적
- 선택 B: 세레나 신뢰 테스트 / 독자 수사

### 노드 8. 공개 의전
- 상황: 대중 앞에서 정통성 선전전.
- 등장인물: 주인공, 세레나, 귀족단
- 핵심 대사
  - 세레나: “오늘은 이기는 게 아니라, 기억되는 거야.”
- 선택 A: 보수 의전 준수 / 파격 상징 연출
- 선택 B: 라이벌과 공동 입장 / 단독 입장

### 노드 9. 서고의 봉인
- 상황: 봉인 서고 침입 흔적. 내부 배신 의심.
- 등장인물: 주인공, 루시안, 엘리안
- 선택 A: 경비 강화 / 내부 미끼 작전
- 선택 B: 엘리안과 강행 / 루시안과 정밀 분석

### 노드 10. 첫 공판
- 상황: 공식 재판 시작. 정치 수치 대량 변동 구간.
- 등장인물: 주인공, 3인 전원, 대중
- 핵심 대사
  - 엘리안: “네가 흔들리면 칼끝도 흔들린다.”
  - 루시안: “증거는 감정 없이 말해.”
  - 세레나: “여론은 증거보다 빨리 달린다.”
- 선택 A: 강경 기소 / 조건부 타협
- 선택 B: 공개 전략 / 비밀 협상

### 노드 11. 왕실색 레시피
- 상황: 왕실색 제조법 조작 의혹.
- 등장인물: 주인공, 루시안
- 선택 A: 문서 전면 공개 / 핵심만 공개
- 선택 B: 레시피 복원 우선 / 범인 추적 우선

### 노드 12. 길드 분열
- 상황: 길드 내 내분 폭발.
- 등장인물: 주인공, 세레나, 길드 간부
- 선택 A: 중립 조정 / 한 편 공개 지지
- 선택 B: 경제 안정 우선 / 정통성 우선

## ACT 3 — 루트 고정 전야 (노드 13~18)

### 노드 13. 라이벌의 제안
- 상황: 공개 적대를 접고 동맹 제안.
- 등장인물: 주인공, 세레나
- 선택 A: 즉시 수락 / 조건부 수락 / 거절
- 선택 B: 비밀 동맹 / 공개 동맹

### 노드 14. 신판 재개
- 상황: 종교·법·민심 충돌.
- 등장인물: 주인공, 루시안, 보수파
- 선택 A: 신 앞의 맹세 / 사람 앞의 맹세
- 선택 B: 보수파 설득 / 개혁파 결집

### 노드 15. 루트 확정 의식
- 상황: 감정선과 정치선이 교차하며 루트 잠금.
- 등장인물: 주인공, 3인 전원
- 선택 A: 질서(엘리안) / 구원(루시안) / 실리(세레나)
- 선택 B: 고백 반걸음 / 정치적 거리 유지

### 노드 16. 봉인의 균열
- 상황: 봉인장치 파괴 시도.
- 등장인물: 선택 루트 상대 + 주인공
- 선택 A: 물리 저지 / 역추적
- 선택 B: 증거 회수 / 인명 보호

### 노드 17. 밤의 편지
- 상황: 정체불명 편지 3통 도착.
- 등장인물: 주인공, 루트 상대
- 선택 A: 편지 공개 / 비밀 보관
- 선택 B: 함정 응답 / 무응답

### 노드 18. 무도회의 함정
- 상황: 춤과 정치가 맞물린 함정 연출.
- 등장인물: 주인공, 루트 상대, 라이벌
- 선택 A: 공개 연기 / 비공개 유인
- 선택 B: 질투 유도 / 안정적 마무리

## ACT 4 — 내란 직전 (노드 19~24)

- 선택 B: 감시도 관리 / 즉시 승부

### 노드 20. 중간보스 재판

- 등장인물: 주인공, 피고, 루트 상대
- 선택 A: 증거 소모 기소 / 증거 보존 우회
- 선택 B: 군중 설득 / 판관 설득

### 노드 21. 내란 징후
- 상황: 도시 외곽 폭동 전조.
- 등장인물: 주인공, 루시안 라인 강화

### 노드 22. 검은 인장

- 등장인물: 주인공, 엘리안, 세레나

- 선택 B: 내부 숙청 / 외부 공세

### 노드 23. 균열의 고백
- 상황: 루트 상대가 숨긴 진실 고백.
- 등장인물: 주인공, 루트 상대
- 선택 A: 용서 / 조건부 수용 / 단절
- 선택 B: 진실 공유 / 비밀 봉인

### 노드 24. 왕관의 무게
- 상황: 권력 승계 문서에 서명 압박.
- 등장인물: 주인공, 귀족원
- 선택 A: 즉시 서명 / 보류

## ACT 5 — 최종 전야와 결말 (노드 25~30)

- 등장인물: 주인공, 루트 상대, 대중

- 등장인물: 주인공, 루트 상대

- 상황: 치명적 진실을 공개할지 판단.
