# DESIGN_V2 — 《보랏빛 규율: 은실로 짠 왕관》

## 코어 루프
1) 스토리 노드 진행
2) 선택으로 5대 수치 변동(정통성/경제력/민심/군사신뢰/감시도)
3) 아르바이트/상점/데이트로 수치 보정
4) 재판/정치 이벤트로 분기 잠금/해금
5) 30노드 도달 후 엔딩 판정

## 30노드 단계 구조
- 1~10: 생존 단계
- 11~20: 권력 전환 단계
- 21~30: 왕권 결정 단계
- 노드 길이: 5~10분

## 수치 규칙
- 범위: 0~100
- 평균 노드 변동: ±5
- 큰 선택: ±15
- 미니게임: ±3~8
- 감시도 100 = 즉시 처벌 엔딩 트리거

## 핵심 시스템
### 복식법(사치금지법)
- 착용 아이템이 직접 수치에 영향
- 위반 장식/왕실색 위조는 감시도 상승

### 4대 축 + 감시도
- 정통성: 의전/왕실색/법정 승리
- 경제력: 길드/시장/재정
- 민심: 은등회/도시 여론
- 군사신뢰: 기사단/전선
- 감시도: 복식 위반/정치적 표적화

### 연애 루트
- 15노드에서 잠정 확정
- 루트별 엔딩 성격 변화
  - 기사단장: 공동통치
  - 길드 수장: 경제왕권
  - 은등회 원장: 시민규율 개혁
  - 라이벌 귀족: 귀족합의형 왕권

### 재판 시스템
- 신판(의식) vs 증거재판 선택
- 공개 법정 + 비밀 신판 이중구조

## 엔딩 그룹(총 12)
- 공동통치 3
- 혁명 2
- 타협 3
- 배드엔딩 4

## UI 원칙
- 메인: 단일 배경 + 주인공 전신 + 오버레이 패널
- 스토리맵: 30노드 세로 스크롤, 하->상, 분기 연결
- 스토리씬: 배경/스탠딩/대화창/선택지

## 반복 플레이
- 2회차: 보랏빛 기억(선택 결과 미리보기)
- 숨은 루트/숨은 레시피 해금
