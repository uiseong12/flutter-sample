# Flutter Sample - 로열 하트 크로니클 (비주얼 노벨형)

요청사항 기반으로 게임형 구조로 확장한 샘플입니다.
모든 배경/캐릭터/의상 이미지는 `assets/art/`의 로컬 SVG(중세풍 2D 일러스트 팩)로 제공됩니다.

## 핵심 기능
- 홈 허브 화면 + 메인 메뉴
  - 스토리 / 아르바이트 / 상점 / 데이트
- 주인공 외형 변화
  - 의상 장착 시 로컬 SVG 아바타 이미지 변경
  - 매력(Charm) 수치 보정
- 스토리
  - 다중 캐릭터 관계 얽힘
  - 선택지마다 메인/서브 호감도 변동
- 아르바이트 미니게임
  - 10초 클릭 챌린지로 골드 획득
- 상점
  - 의상 구매 및 착용
  - 호감도 상승 아이템 구매/선물
- 데이트
  - 호감도 구간 기반 랜덤 이벤트
- 로그/저장
  - shared_preferences로 상태 저장

## 실행
```bash
flutter pub get
flutter run -d chrome
```
