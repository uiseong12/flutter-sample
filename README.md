# Flutter Sample - 중세 로맨스 스토리 앱 MVP+

중세 여성 주인공 기반의 스토리형 연애 시뮬레이션 앱입니다.

## 포함 기능

- 2명 캐릭터(엘리안, 루시안)
- 캐릭터 이미지 표시(브라우저 기반 URL 이미지)
- 호감도 시스템 (0~100)
- 스토리 선택지 이벤트 (JSON 에셋 분리)
- 데이트 기능
- 선물하기 기능
- 이벤트 로그
- 엔딩 분기(호감도 기반)
- 주인공 장비 시스템(아이템 장착/해제)
- 아이템 기반 매력 수치 보정(호감도 증가량 가중)
- 진행 상태 저장/복구 (`shared_preferences`)
- 진행 초기화(앱바 리셋 버튼)

## 실행 방법

```bash
flutter pub get
flutter run
```

## 구조

- `lib/main.dart`
  - 스토리/관계/이벤트 탭
  - 캐릭터 상태(호감도/데이트 횟수/선물 횟수)
  - 저장/복구 로직
- `assets/story_events.json`
  - 스토리 이벤트 데이터
