# Flutter Sample - 로열 하트 크로니클 (비주얼 노벨형)

요청사항 기반으로 게임형 구조로 확장한 샘플입니다.
모든 배경/캐릭터/의상 이미지는 `assets/art/`의 로컬 SVG(중세풍 2D 일러스트 팩)로 제공됩니다.

## 핵심 기능
- 홈 허브 화면 + 메인 메뉴
  - 스토리 / 아르바이트 / 상점 / 데이트
- 주인공 외형 변화
  - 의상 장착 시 전신 외형 변경
  - 매력(Charm) 수치 보정
- 스토리 (비주얼노벨형)
  - 배경 + 전신 캐릭터 좌우 배치 + 하단 대화창
  - 선택지 버튼에 캐릭터명 표기 (예: [엘리안] ...)
  - 다중 캐릭터 관계 변동
- 엔딩 규칙
  - 각 캐릭터 중 호감도 100을 **가장 먼저** 달성한 캐릭터 엔딩 확정
- 게임 연출 강화
  - 페이지 전환 Fade/Scale
  - 스토리 장면 전환 Preset (fade/slide/flash)
  - 캐릭터 등장/퇴장 슬라이드 + 감정(표정) 상태 컬러 연출
  - 대사 타이핑(Typewriter), Auto/Skip
  - 클릭/보상 시스템 사운드 훅
  - 관계도 변화량(+N) 팝업, 스파클 파티클
  - 카메라 줌/팬 느낌의 배경 Tween
  - 목표 패널 + 감정/무드 오버레이
- 아르바이트 미니게임
  - 10초 클릭 챌린지로 골드 획득
- 상점
  - 의상 구매/착용
  - 호감도 상승 아이템 구매/선물
- 데이트
  - 호감도 구간 기반 랜덤 이벤트
- 로그/저장
  - shared_preferences로 상태 저장

## 실행
```bash
flutter pub get
flutter run -d chrome
```
