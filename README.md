# Flutter Sample - 중세 로맨스 스토리 앱 MVP

중세 여성 주인공 기반의 스토리형 연애 시뮬레이션 MVP입니다.

## 포함 기능 (MVP)

- 2명 캐릭터(엘리안, 루시안)
- 호감도 시스템 (0~100)
- 스토리 선택지 이벤트
- 데이트 기능
- 선물하기 기능
- 이벤트 로그

## 실행 방법

```bash
flutter pub get
flutter run
```

## 구조

- `lib/main.dart`
  - 스토리/관계/이벤트 탭
  - 캐릭터 상태(호감도/데이트 횟수/선물 횟수)
  - 선택지에 따른 호감도 증감
