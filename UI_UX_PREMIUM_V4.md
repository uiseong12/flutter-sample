# UI/UX + Premium Integration V4

## 반영 요약
- 프리미엄 선택지 20개 샘플 데이터 파일 추가: `premium_choices_v1.json`
- 프리미엄 선택지 오픈 플로우(토큰/광고/나중에) UI 구현
- 스토리 대화 선택지 3단 계층(무료/조건/프리미엄 라벨) 반영
- 노드맵 아이콘 분류(감정/재판/의전/조사) + 노드 프리뷰 바텀시트 반영
- 홈 화면 상단 재화/AP + 오늘의 추천 카드/다음 노드 바로가기 반영
- IA 확장: 홈/스토리/데이트/아르바이트/제작상점/장부/도감/설정
- 장부 화면(정치수치 막대, 증거 카드, 최근 로그) 추가
- 도감 화면(보이스/CG/엔딩/POV 진행형) 추가
- 설정 화면(오토/스킵/광고 원칙 문구) 추가

## 광고/과금 UX 원칙
- 감정씬 직전/직후 강제 광고 금지
- 보상형 광고 중심
- 프리미엄 선택지는 토큰 또는 광고로 오픈 가능
- 노드 결과 화면에서 광고 기반 추가 보상/토큰 획득 제공
